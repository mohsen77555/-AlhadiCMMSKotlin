Suggested groups:

1. Work and maintenance
   - Notifications
   - Preventive maintenance
   - Task lists
   - Meters
   - Failure analysis
   - CAPA

2. Assets and locations
   - Assets
   - Functional locations
   - Asset documents
   - Asset characteristics
   - Asset BOM

3. Inventory and procurement
   - Inventory
   - Procurement
   - Suppliers

4. Governance and administration
   - Audit log
   - Trash
   - Admin and users
   - Backup and restore

5. Reports
   - Reports
   - PDF exports

Do not create one separate tab for every internal rule. Internal rules should affect behavior, messages, validation, and audit logging. The visible UI should stay practical and grouped by work flow.
